### Reticulated Python

Michael Vitousek 的 Reticulated Python 是为 Python 进行渐进式类型检查的略有不同的方法的一个示例。它在 Vitousek 与 Jeremy Siek 和 Jim Baker（后者的 Jython 闻名）撰写的实际学术论文中描述。

### PyCharm

JetBrains 的 PyCharm 已经提供了指定和检查类型的方法约四年。PyCharm 建议的类型系统从简单的类类型发展到元组类型、泛型类型、函数类型等，基于许多用户分享他们在代码中使用类型提示的经验反馈。

## 其他语言中的现有方法

### ActionScript

ActionScript 是基于类的、单继承的、面向对象的 ECMAScript 超集。它支持接口和强运行时检查的静态类型。编译支持"严格方言"，其中类型不匹配在编译时报告。

带有类型的示例代码：

```actionscript
package {
    import flash.events.Event;

    public class BounceEvent extends Event {
        public static const BOUNCE:String = "bounce";
        private var _side:String = "none";

        public function get side():String {
            return _side;
        }

        public function BounceEvent(type:String, side:String){
            super(type, true);
            _side = side;
        }

        public override function clone():Event {
            return new BounceEvent(type, _side);
        }
    }
}
```

### Dart

Dart 是基于类的、单继承的、面向对象的语言，具有 C 风格的语法。它支持接口、抽象类、具体化泛型和可选类型。

类型在可能时推断。运行时区分两种执行模式：旨在用于开发的检查模式（在运行时捕获类型错误）和推荐用于快速执行的生产模式（忽略类型和断言）。

带有类型的示例代码：

```dart
class Point {
    final num x, y;

    Point(this.x, this.y);

    num distanceTo(Point other) {
        var dx = x - other.x;
        var dy = y - other.y;
        return math.sqrt(dx * dx + dy * dy);
    }
}
```

### Hack

Hack 是一种与 PHP 无缝互操作的编程语言。它提供可选的静态类型检查、类型别名、泛型、可空类型和 lambda。

带有类型的示例代码：

```hack
<?hh
class MyClass {
    private ?string $x = null;

    public function alpha(): int {
        return 1;
    }

    public function beta(): string {
        return 'hi test';
    }
}

function f(MyClass $my_inst): string {
    // Will generate a hh_client error
    return $my_inst->alpha();
}
```

### TypeScript

TypeScript 是 JavaScript 的已类型超集，它为语言添加了接口、类、混入和模块。

类型检查是鸭子类型。通过提供重载函数声明来指定多个有效函数签名。函数和类可以使用泛型作为类型参数化。接口可以有可选字段。接口可以指定数组和字典类型。类可以有构造函数，隐式地将参数添加为字段。类可以有静态字段。类可以有私有字段。类可以有字段的 getter/setter（像 property）。类型被推断。

带有类型的示例代码：

```typescript
interface Drivable {
    start(): void;
    drive(distance: number): boolean;
    getPosition(): number;
}

class Car implements Drivable {
    private _isRunning: boolean;
    private _distanceFromStart: number;

    constructor() {
        this._isRunning = false;
        this._distanceFromStart = 0;
    }

    public start() {
        this._isRunning = true;
    }

    public drive(distance: number): boolean {
        if (this._isRunning) {
            this._distanceFromStart += distance;
            return true;
        }
        return false;
    }

    public getPosition(): number {
        return this._distanceFromStart;
    }
}
```
